Cross-Sectional Templates: Barplot

The below creates a barplot of the frequencies of different categories.

First we create a data set (data_list) comprising 3 values of category 'a', 5 of 'b', 2 of 'c' and 6 of 'd':

In [17]:
#Following code to set up data:

data_list = []

data_list += (3 * 'a ').strip().split(' ')

data_list += (5 * 'b ').strip().split(' ')

data_list += (2 * 'c ').strip().split(' ')

data_list += (6 * 'd ').strip().split(' ')

Bar charts comprise of an index (the values on the x axis) and values (the values on the y axis - i.e. the frequencies).

It is best to use a np.arange() to set the index value, as then elementwise addition can be used when determining where the x-axis labels are placed. Without these labels, the index values (0, 1, ...) are used instead.

In [18]:
import numpy as np # To enable  numpy arrays, and so element-wise addition

#First we want the categories - the unique values in our list:
categories = ['a', 'b', 'c', 'd']

#The above can be found for a pandas Series using the function pd.Series.unique()

#Second we want the frequencies:
values = [data_list.count(category) for category in categories]

#The index can be found as follows:
index = np.arange(len(categories))

#We can add our category labels as follows:
plt.xticks(index+0.4, categories)

#We can change the graph's dimensions:
plt.axis([-0.2, 4, 0, 7])

#To plot the bar chart:
plt.bar(index, values, color = 'k')

plt.show()

Instead, a horizontal bar chart can be created, if plt.barh() is used instead of plt.bar(). Then, xticks must be replaced by yticks

Contour Plot

In [19]:
#Following requires both NumPy and Pyplot.

#To set the x and the y axes:
x = np.arange(-10,10)
y = np.arange(-10,10)

X,Y = np.meshgrid(x,y)

#The function whose contour plot is being found:
def f(x,y):
    return (x**2 + y**2)/10.0

#Drawing the contour lines:
CS = plt.contour(X, Y, f(X,Y), 10, colors='black') #colors sets contour lines' colours

#Filling in the contour plot with colour:
plt.contourf(X,Y,f(X,Y),10, cmap=plt.cm.hot) #cmap argument leads a heat map to be generated

#Note: the number '10' determines the number of contour lines and the gradations in the colour.

#Adding labels:
plt.clabel(CS, inline=1, fontsize=10, colors = 'blue') #the inline argument causes labels to be written over the lines

#Adding colour scale:
plt.colorbar() 

plt.show()

Above contour plot code based on:

  • Nelli, F., 2015. "Python Data Analytics: Data Analysis and Science Using Pandas, Matplotlib, and the Python Programming Language". Apress. Available as an ebook

The Codes for different Lines and Colours

The different colours can be summarised as follows:

  • b: blue
  • g: green
  • r: red
  • c: cyan
  • m: magenta
  • y: yellow
  • k: black
  • w: white

Further information on a wider variety of colours is available in the documentation
